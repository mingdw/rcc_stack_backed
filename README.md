# 蓝宝石代币质押商城系统

蓝宝石代币质押商城系统是一个基于区块链技术的代币质押商城系统，用户可以通过质押代币来获取代币，并且通过代币可以兑换商场里面的虚拟商品，交易方便快捷，安全可靠。

[前端源码](https://github.com/mingdw/rcc-stake-fe-react.git)

![特别鸣谢：Nunu](https://github.com/go-nunu/nunu/blob/main/.github/assets/banner.png)
